# Skills for Hire – Backend

## Description
Skills for Hire is a backend application that powers a service marketplace where skilled professionals can offer their services and clients can hire them. The backend provides RESTful APIs for user authentication, skill listings, job requests, hiring workflows, and reviews.

This project is built with Django and Django REST Framework

---

## Features
- User registration and authentication (JWT)
- Role-based users (Clients and Service Providers)
- Skill and service listings
- Job posting and job applications
- Hiring and job status management
- Reviews and ratings
- Admin management

---

## Technologies Used
- Python
- Django
- Django REST Framework (DRF)
- JWT Authentication (SimpleJWT)
- PostgreSQL
